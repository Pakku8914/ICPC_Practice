# 答えとの参照方法
1. プロジェクトを右クリし、プロパティを開く
2. デバッグのコマンド引数に`< in.txt > out.txt`を記述する
3. プロジェクト下のソースファイルディレクトリに`in.txt`, `out.txt`を追加する
4. プログラム実行
5. out.txtに出力された事項を確認
6. VScodeで`ctrl + shift + p`によってコマンドパレットを開く
7. `ctrl + d`でアクティブされたファイルの比較を実行
8. 比較対象のファイルを選択する
